# Saw Text Adventure

This repository contains a command-line text adventure set in the world of
**Saw**. Your goal is to solve puzzles, make moral choices, and escape the
twisted trials laid out by Jigsaw.

## Getting Started

Run the adventure with Python 3.8 or later:

```bash
python saw_text_adventure.py
```

## Gameplay

* Type commands to interact with the environment (e.g., `look`, `inspect door`).
* Use `help` to see the complete list of available commands.
* Examine everything carefully—light, time, and compassion matter.
